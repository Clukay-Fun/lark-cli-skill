# v1.0.0 - Initial Release

Community extension skill pack for scenarios not covered by the official `larksuite/cli` skills.
Supports Claude Code, Codex, OpenCode, and Gemini CLI environments.

## Included Skills

### Environment

- `lark-setup` - one-entry setup flow that detects the active CLI environment, installs `lark-cli`, installs the official skill pack, installs these extra skills, and checks authentication status

### Domain Extensions

- `lark-sheets-extra` - Sheets v2 helpers for merge and unmerge, styling, and row or column operations
- `lark-base-batch` - Bitable batch create, update, and delete operations, up to 500 records per request
- `lark-im-card` - interactive card authoring guide with reusable templates

### Workflows

- `lark-workflow-meeting` - attendee lookup, availability check, event creation, and notification flow
- `lark-doc-convert` - unified local file and Lark document import and export flow, including timeout handling and wiki link resolution

## Installation

Say this to your AI CLI agent:

> Install lark-cli and the extra Lark skills for me.

Or use the manual installation steps in [`README.md`](../README.md).

## Requirements

- Node.js 16 or later
- The official `larksuite/cli` skill pack, which `lark-setup` can install automatically

## Notes

- This repository does not vendor the official skills. It only provides supplemental docs and helper scripts.
- Batch delete operations in `lark-base-batch` still require explicit confirmation before execution.
- `lark-cli auth login` may require interactive user participation.
