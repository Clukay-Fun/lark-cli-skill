# Card Element Quick Reference

Chinese translation: [card-elements.zh-CN.md](card-elements.zh-CN.md)

## `markdown`

Use for Markdown text blocks.

## `div`

Use for a text block or short-field layout.

## `column_set` and `column`

Use for multi-column layouts with more control than `div.fields`.

## `action`

Use for buttons.

## `note`

Use for muted footer information.

## `img`

Use for uploaded images referenced by `img_key`.
